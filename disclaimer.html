Disclaimer - WorldHams App

This app collects the following data:

  • Name, Callsign, Handle, Location
  • Contact Number (optional), Email, Password
  • Profile Photo(optional) and Location Coordinates

Publicly(only registerd users in this app) visible data:

  • Name, Callsign, Handle, Antenna Height, Location Name

Never shared publicly:

  • Email, Password, Contact Number, Coordinates, Elevation

Permissions:

  • Location: Fetched once at signup to map exact QTH.
  • Camera & Gallery: For optional profile photo at signup.

After using the app, users are advised to disable permissions from system settings.

Storage:

All data is stored securely on cloud servers. Only registered users can view non-sensitive data of other hams.

By signing up, users confirm they are licensed amateur radio operators and agree not to misuse the data. Legal liability for misuse rests with the user.
