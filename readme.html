WorldHams

This app is Amateur Radio Related.

Amateur Radio Operators are known worldwide as "Ham". This is a worldwide hobby.

This hobby requires a licence from the Central Government of the country of residence to operate on wireless equipment.

This app asks for:

  • Callsign of Ham (globally unique)
  • Name, Handle (short nickname), Location (city/village)
  • Antenna Height, Contact Number
  • User Photo - via camera or gallery)
  • Email (verified at signup), Password

Location coordinates and elevation are fetched automatically with permission during signup only.

Once logged in, users see:

  • Ham List View with Callsign and Name
  • Card View with full Ham details including photo
  • Searchable List and Map View showing user locations
  • Distance and bearing between stations
  • Elevation graphs, Fresnel Zone, Radio Horizon prediction

Privacy Policy:

Email is verified at signup to avoid fake users. Password is securely stored (encrypted). Location data is used only in-app and not shared to Google Maps publicly.
